# atomic-mixing
This program calculates the atomic mixing due to ion-induced cascades simulated using the TurboGAP code.
The user can provide inputs interactively, which are self-explanatory, to use the program.
Reference work: "Improved capabilities of the TurboGAP code for radiation induced cascade simulations: An illustration with silicon". Computational Materials Science 267 (2026) 114560. https://doi.org/10.1016/j.commatsci.2026.114560
